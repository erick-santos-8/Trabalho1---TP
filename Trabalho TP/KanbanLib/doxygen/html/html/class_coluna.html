Detailed Description

Coluna é um domínio que tem como papel definir a localizacao de um cartao no sistema, ela pode ser: "SOLICIDATADO", "EM EXECUCAO", "CONCLUIDO".

Note
Especificação
Deve ter valor = "SOLICITADO", "EM EXECUCAO" ou "CONCLUIDO".
